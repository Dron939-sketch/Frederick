## 🚀 Возможности

| Модуль | Описание |
|--------|----------|
| 🧠 **AI-психолог** | На базе DeepSeek API, с тремя режимами общения |
| 🎤 **Голосовой интерфейс** | STT (Deepgram) + TTS (Yandex SpeechKit) |
| 📊 **Психологическое тестирование** | 4 вектора, 9 элементов, глубокая аналитика |

| 💭 **Мысли психолога** | Персональные инсайты и рефлексия |
| 🎯 **Персональные цели** | Индивидуальные рекомендации и трекинг |
| 📚 **Терапевтические сказки** | Метафоры для работы с подсознанием |
| ⚓ **Гипнотические якоря** | Установка и активация ресурсных состояний |
| 🌤 **Погода и контекст** | Адаптация под пользователя |
| 📱 **Адаптивный дизайн** | Для всех устройств |
