# Фреди - Виртуальный психолог

Премиум веб-приложение с AI-психологом, голосовым интерфейсом и глубокой аналитикой личности.

## 🚀 Возможности

- 🧠 **AI-психолог** на базе DeepSeek API
- 🎤 **Голосовой интерфейс** (STT + TTS) через Deepgram и Yandex SpeechKit
- 📊 **Психологическое тестирование** (4 вектора, 9 элементов)
- 🔐 **Модель ограничений** (Confinement Model)
- 💭 **Мысли психолога** - персональные инсайты
- 🎯 **Персональные цели** и рекомендации
- 📚 **Терапевтические сказки**
- ⚓ **Гипнотические якоря**
- 🌤 **Погода** и контекст пользователя
- 📱 **Адаптивный дизайн** для всех устройств
